luftrom.info: Resources for paraglider pilots in Norway

This repository contains several data files than can be useful for planning XC trips in Norway.

Airspace

This is best effort airspace information geared towards paraglider pilots.
THESE FILES ARE PROVIDED WITHOUT WARRANTY OF ANY KIND. Errors and omissions will occur, and updates happen on a best effort basis. Each geometry contains a data source link which you *should always* check for possible updates. See also madsen-system.se/luftrum for updated Swedish airspace information by local glider pilots.

Please consult also the change log (Norwegian) for inclusion or omission of updates. Subscribe to be notified of changes (via ATOM feeds) here.

Files:

  • Airspace in geojson format
  • OpenAIR format in meters and feet above MSL - skipping airspace above 3500m
  • OpenAIP format (for inclusion in openaip.net)

Other resources

In no particular order:

  • Aggregate over all flown routes from the Norwegian flightlog
  • Terrain model showing slope values
  • Registered takeoffs from flightlog.org
  • Public cabins from Kartverket N50 data
  • Mountain tops from Kartverket N50 data
  • Surfaces of water from Kartverket N250 data
  • Marked trails and ski tracks for Norway
  • Some thermal hotspots extracted from flightlog.org
  • Nasjonalt register over Luftfartshindre (Kartverket)
  • Locations of model flying airports in Norway (NLF).
